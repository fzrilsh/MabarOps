Cara Kerja
1.
Aktifkan Sesi

Streamer toggle ON. Sistem listen donasi Sawiera.

2.
Donasi Masuk

Penonton donasi + keyword "mabar". Auto-masuk antrean.

3.
Moderator Buat Room

Moderator buat room, invite streamer, publish code.

4.
Mabar Dimulai

Pelanggan join pakai code, main bareng, lalu rating.

Fitur Utama
Parser Donasi Otomatis

Integrasi WebSocket Sawiera. Deteksi nominal & keyword mabar. Fast-track vs Regular otomatis terbedakan.

Smart Queue 2 Jalur

Antrean Regular & Fast-track terpisah. Prioritas berdasarkan level prioritas yang ditentukan streamer.

OBS Overlay

Tampilkan antrean di siaran via Browser Source. Real-time, full-screen, readable di background apapun.

Portal Pelanggan Permanen

Unique link permanen via email. Atur Game ID, lihat room code real-time, beri feedback selesai main.
